PREVIEW PANEL

OPTIONAL FUTURE ARTWORK
The code automatically looks for:
- atmousepheres-morning.png
- atmousepheres-night.png
If either file is absent, the approved atmousepheres-final.png remains in place and CSS lighting effects create the time treatment. This means the build works immediately without broken images.
